스크린 사이즈는 max-width: 800px 

@media screen and (max-height: 700px) { 
	margin-bottom: 80px ;
}
@media screen and (min-height: 701px) { 
	margin-bottom: 11.4vh
}
화면 높이가 700px 를 충족하지 못하는 경우 margin-bottom를 80px 적용 만약 아니라면 11.4vh 
@media (min-width: 601px){
	margin-left: 40px;
	margin-right: 40px;
}

@media (min-width: 401px) and (max-width: 600px) {
	margin-left: calc(20px + (100vw - 400px) / 10);
	margin-right: calc(20px + (100vw - 400px) / 10);
}

@media (max-width: 400px) {
	margin-left: 20px;
	margin-right: 20px;
}
화면 margin 설정

Card와 Code의 경우 아래의 margin를 제거하여 좀더 넓은 화면을 제공

margin-left: -10px;
margin-right: -10px;
